# NextUpTV ΓÇõ Portfolio Documentation Index

This suite of documents describes the design, build, and hardening of **NextUpTV**, an AI-powered TV show recommendation platform built as a portfolio project to demonstrate applied AI engineering skills.

The project was built using Claude Sonnet 4.6, Next.js 16, and TypeScript. It includes streaming AI output, an LLM-as-judge evaluation framework, prompt injection hardening, and per-request cost tracking.

## The One-Paragraph Summary

NextUpTV is a full-stack web application that takes a user's watch history (uploaded as a file or pasted as keywords) and returns personalised TV show recommendations generated by Claude Sonnet 4.6. Recommendations are streamed progressively to the browser using Server-Sent Events, with TVDB metadata lookups (posters, genres, streaming platforms) running in parallel with the AI generation. The project includes a built-in LLM-as-judge evaluation framework that ran 27 scored experiments across prompt iterations, moving from an initial grade of C (7.7) to a stable B (8.4ΓÇô8.5). All AI inputs are hardened against prompt injection. Every production request is logged with token counts, cost in USD, and geo-location.
