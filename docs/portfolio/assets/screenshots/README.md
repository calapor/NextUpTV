## Tips

- Use a browser window at 1440px wide for desktop shots (Command+Shift+5 on Mac, or browser DevTools device toolbar)
- For mobile, use Chrome DevTools → Toggle device toolbar → iPhone 12 Pro preset
- Crop to the relevant UI area — no need to show the full browser chrome
- PNG format preferred; keep files under 500KB where possible (use TinyPNG if needed)
